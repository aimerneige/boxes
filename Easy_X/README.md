# Boxes with Easy_X

使用 Easy_X 图形库构建的推箱子小游戏。

图片关卡等资源均来源于网络，仅供学习交流使用。

## How to build

- 下载并安装 [Visual Studio 2019 Professional](https://visualstudio.com)
- 下载并安装 [Easy_X](https://easyx.cn/)
- 双击本文件夹下的 `Box_AimerNeige.sln`
- 等待 VS 将配置加载完成后，点击绿色三角形即可构建应用
